Getting Started

You just saw the working area of graph app. To add nodes press your mouse onto it.

Controls

  • Press left mouse button on a free space - add node
  • Press left mouse button on a free space and drag - selection region
  • Press right mouse button on a free space and drag - move canvas
  • Mouse wheel button - zoom in / out
  • Ctrl + Click - select one item
  • N - create new edge between two nodes
  • Delete - delete selected elements (both nodes and edges)
  • F2 - rename node (if node selected), reweight edge (if edge selected)
